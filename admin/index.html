Base de dados partilhada (nuvem)

Base de dados Firebase incorporada directamente no código. Sincronização automática activa a cada 30 segundos.

Para mudar a base de dados, edite o ficheiro index.html e substitua o valor de _DB_URL pelo novo endereço. Esta é a única forma de o fazer — não existe formulário de configuração para evitar que utilizadores não autorizados alterem a ligação.
